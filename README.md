Excited to share my latest project: A Sleep Quality

Predictor & Visualizer!🌙📊

In a world where rest is often undervalued, I wanted to build a tool that helps quantify and visualize how our daily choices-like screen time, exercise, and stress-directly impact the quality of our sleep.

Using Python and Streamlit, I developed an interactive web application that:

💻Calculates a personalized Sleep Score based on key lifestyle inputs.

📌Visualizes the data using Matplotlib to provide a clear, graphical representation of these habits.

💻Provides instant feedback through a sleek, user-friendly interface.

This project was a great way to combine data logic with front-end deployment, showing how simple code can turn personal habits into actionable insights.
